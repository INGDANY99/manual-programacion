Condicionales en Java

Java permite tomar decisiones mediante estructuras como if, else y switch.


int edad = 20;
if (edad >= 18) {
    System.out.println("Eres mayor de edad");
} else {
    System.out.println("Eres menor de edad");
}

// Switch
int dia = 3;
switch (dia) {
  case 1:
    System.out.println("Lunes");
    break;
  case 2:
    System.out.println("Martes");
    break;
  default:
    System.out.println("Otro día");
}

Ciclos en Java

Java incluye varios tipos de bucles para repetir acciones.


// Bucle for
for (int i = 0; i < 5; i++) {
    System.out.println(i);
}

// Bucle while
int i = 0;
while (i < 5) {
    System.out.println(i);
    i++;
}

Funciones en Java

En Java las funciones se llaman "métodos" y están dentro de clases.


public class Main {
  public static void saludar(String nombre) {
    System.out.println("Hola, " + nombre);
  }

  public static void main(String[] args) {
    saludar("Biggie");
  }
}
