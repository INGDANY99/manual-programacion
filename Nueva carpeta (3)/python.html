Condicionales en Python

Las estructuras condicionales permiten ejecutar código dependiendo de una condición.


if edad >= 18:
    print("Eres mayor de edad")
else:
    print("Eres menor de edad")

Ciclos en Python

Se utilizan para repetir tareas. Python ofrece for y while.


# Bucle for
for i in range(5):
    print(i)

# Bucle while
i = 0
while i < 5:
    print(i)
    i += 1

Funciones en Python

Las funciones permiten agrupar código que puede reutilizarse.


def saludar(nombre):
    print(f"Hola, {nombre}!")

saludar("Tupac")
