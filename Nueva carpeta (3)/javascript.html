Condicionales en JavaScript

JavaScript permite tomar decisiones con if, else y switch.


let edad = 20;

if (edad >= 18) {
  console.log("Eres mayor de edad");
} else {
  console.log("Eres menor de edad");
}

// Switch
let dia = 3;
switch (dia) {
  case 1:
    console.log("Lunes");
    break;
  case 2:
    console.log("Martes");
    break;
  default:
    console.log("Otro día");
}

Ciclos en JavaScript

Los bucles permiten repetir tareas.


// For
for (let i = 0; i < 5; i++) {
  console.log(i);
}

// While
let i = 0;
while (i < 5) {
  console.log(i);
  i++;
}

Funciones en JavaScript

JavaScript permite declarar funciones para reutilizar código.


function saludar(nombre) {
  console.log("Hola, " + nombre);
}

saludar("Nas");
