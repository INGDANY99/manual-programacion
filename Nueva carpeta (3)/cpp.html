Condicionales en C++

Permiten ejecutar bloques de código según condiciones lógicas.


int edad = 18;

if (edad >= 18) {
    std::cout << "Eres mayor de edad" << std::endl;
} else {
    std::cout << "Eres menor de edad" << std::endl;
}

Ciclos en C++

Usamos for, while y do-while para repetir instrucciones.


// For
for (int i = 0; i < 5; i++) {
    std::cout << i << std::endl;
}

// While
int i = 0;
while (i < 5) {
    std::cout << i << std::endl;
    i++;
}

// Do-while
int j = 0;
do {
    std::cout << j << std::endl;
    j++;
} while (j < 5);

Funciones en C++

Las funciones permiten estructurar el código y reutilizarlo fácilmente.


#include <iostream>

void saludar(std::string nombre) {
    std::cout << "Hola, " << nombre << std::endl;
}

int main() {
    saludar("Eminem");
    return 0;
}
